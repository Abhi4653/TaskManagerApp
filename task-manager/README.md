# Task Manager

Task Manager is a monorepo containing a frontend and backend for a simple task management application.

## Project Structure
- frontend: Web client (e.g., React/Vite)
- backend: API server (e.g., Node/Express)

## Running locally

Prerequisites: Node.js 18+ and npm.

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend

#### Backend (development)
```bash
cd backend
npm install
npm run dev
```

```bash
cd backend
npm install
npm start
```
